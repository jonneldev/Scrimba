Mastering Frontend Basics with Scrimba

Strengthened my knowledge in frontend web development. Through the Scrimba platform, I learned the fundamentals of HTML, CSS, and JavaScript, while building several mini-projects to put my skills into practice. The interactive coding environment and supportive community made it easier to grasp difficult concepts. I particularly enjoyed creating responsive layouts and implementing modern CSS techniques like Flexbox and Grid. These skills helped me gain confidence in building user-friendly interfaces and applying industry best practices.

Diving into Full-Stack Development at Zuitte

Learned the basics of full-stack development. At Zuitte Bootcamp, I explored the integration of frontend and backend technologies, starting with creating APIs and connecting them to user interfaces. I also gained hands-on experience with Node.js and Express.js for server-side development and MongoDB for database management. This bootcamp emphasized practical application, so I worked on building a simple yet functional CRUD app, which allowed me to see how everything comes together. This experience opened my eyes to the power of full-stack development.

Building Solid Foundations with The Odin Project

Learned basic HTML, CSS, and Git. The Odin Project provided a comprehensive introduction to web development. The Git lessons were particularly insightful, as I learned how to version control my projects effectively, collaborate with others, and deploy projects seamlessly. These foundational skills have set the stage for more advanced learning and reinforced my passion for coding.
